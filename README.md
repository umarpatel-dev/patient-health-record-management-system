# Patient Health Record Management System

This project is a web-based application designed to manage patient health records efficiently. It provides two separate interfaces for patients and doctors.

## Features
- Patient login and registration
- Doctor login and access
- Secure storage of patient medical data
- Easy record management
- User-friendly interface
